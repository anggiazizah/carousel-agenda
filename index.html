Halaman Login

Pertama, user dan admin login terlebih dahulu dengan cara yang berbeda. User login menggunakan NIS dan Password yang sudah disediakan oleh pihak sekolah dan admin login menggunakan email dan password.

Halaman Beranda

Setelah login, user akan dialihkan ke halaman beranda untuk mengisi agenda.

Halaman Agenda

Pada halaman agenda ini, berisi riwayat agenda yang sudah dikerjakan hari ini. Tetapi, user tidak bisa mengubah yang sudah dilakukan di halaman ini.

Halaman Kalender

Pada halaman kalender, user dapat melihat agenda shalat yang di kerjakan/checklist di hari-hari sebelumnya dengan mengklik tanggal yang di inginkan. Lalu akan diarahkan ke halaman agenda yang berisi riwayat pada tanggal yang di klik tersebut

Halaman Profile user

Pada halaman profile, user dapat melihat dan dapat mengubah beberapa data profilnya.

Halaman Dashboard Admin

Setelah admin login, admin akan di arahkan ke halaman dashboard yang berisi informasi singkat mengenai jumlah user yang sudah terdaftar dan juga jumlah siswa yang sudah mengisi agenda shalat di hari itu.

Halaman Agenda Admin

Pada halaman agenda admin, admin dapat melihat siapa saja yang telah mengisi agenda shalat hanya pada hari itu beserta nama-nama usernya.

Halaman Kalender Admin

Pada halaman kalender Admin, admin dapat melihat siapa saja yang sudah mengisi agenda shalat di hari-hari sebelumnya dengan mengklik tanggal yang di inginkan, lalu akan diarahkan ke halaman agenda admin pada tanggal yang diklik tersebut.

Halaman Pengguna

Pada halaman Pengguna ini admin dapat mengelola data siswa.
